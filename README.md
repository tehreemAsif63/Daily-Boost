# Daily-Boost
A simple web application that delivers random motivational messages to users.

## Features
- Spring Boot backend with a REST API endpoint `/motivation`
- Returns a random motivational message
- Simple HTML + CSS frontend
- Button on the page fetches a new message from the backend
- Fully functional demo-ready in under 2 hours

## Tech Stack
- Java
- Spring Boot
- Spring Web
- HTML/CSS
- JavaScript (fetch API for frontend-backend communication)

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the Spring Boot application
4. Open a browser and go to `http://localhost:8080/`
5. Click **Get Motivation** to see a random message
